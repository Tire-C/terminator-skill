# Example: Project Recovery

```text
Use Terminator to inspect this repository, find the main blocker, fix safe issues, and verify that the project can run.
```

Expected orchestration:

1. Scan repository structure.
2. Detect project type and package manager.
3. Discover available build/test commands.
4. Build a Tool Registry.
5. Run safe checks.
6. Identify the main blocker.
7. Apply minimal safe fixes.
8. Verify with build/test/runtime checks.
9. Return a concise report.
