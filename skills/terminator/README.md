# Terminator Skill

Terminator is an environment-adaptive orchestration skill for AI agents.

It turns user goals into multi-tool workflows by discovering and coordinating the tools available in the current environment.

## Invoke

```text
Use Terminator to ...
```

```text
Usa Terminator per ...
```

```text
Terminator
```
